# Changelog

All notable changes to the Papercuts Chrome Extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## Version Format

- **Major (X.0.0)**: Breaking changes, major feature overhauls
- **Minor (0.X.0)**: New features, significant improvements
- **Patch (0.0.X)**: Bug fixes, small improvements

## [1.0.0] - 2025-12-20

### Added
- Initial release
- Screenshot capture (select area or full screen)
- Module selector (CoreHR, Recruit, Perform, Pulse, Time, Desk)
- Rich text description editor with formatting
- Keyboard shortcuts (S for select area, F for full screen, Ctrl+Shift+S to open)
- Image upload to Cloudinary
- NextAuth authentication integration
- Automatic user email tracking

### Features
- Capture screenshots from any webpage
- Add structured feedback with name, module, and description
- Preview screenshots before submitting
- Extension connects to web app via API key
- Version tracking with git hash and date
